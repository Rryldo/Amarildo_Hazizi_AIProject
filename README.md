## Optimal path finder in a road network with time-varying predicted costs - AI Project

## Group Members
- Amarildo Hazizi

## Project Overview

This project implements an **AI-based route planner** that predicts travel times across a network of roads using machine learning

## How to run the Project

### Dependencies
The project uses Python and requires the following packaes:
- pandas
- numpy
- scikit-learn
- heapq
- datetime
- Jupyter Notebook or any Python IDE

You can install the dependencies using pip:
pip install pandas numpy scikit-learn
